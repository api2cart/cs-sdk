API2Cart C# SDK for eCommerce Integration

Overview

The API2Cart C# SDK provides a unified API to interact with multiple eCommerce platforms, such as Shopify, WooCommerce, Magento, PrestaShop, BigCommerce, and more. This SDK simplifies managing products, orders, customers, and other eCommerce data across different platforms using C#.

With the C# SDK, developers can integrate their applications with eCommerce platforms quickly and efficiently, reducing the complexity of maintaining multiple integrations.

Getting Started

Follow these simple steps to get started with the API2Cart C# SDK:

  1. Clone the repository and restore the dependencies:
    2. git clone https://github.com/api2cart/cs-sdk.git
    cd cs-sdk
    dotnet restore
  2. Set your API key in the configuration file.
  3. Use the SDK to make API calls to the platform of your choice.

Example Code

Here's a simple example of how to use the API2Cart C# SDK to make a request to the API: 


using Api2Cart;

// Set your API key
string apiKey = "YOUR_API2CART_KEY";

// Initialize the API2Cart client
var client = new Api2CartClient(apiKey);

// Make a request to check the API status
var response = client.Status();

// Print the response
Console.WriteLine(response);

Replace `'YOUR_API2CART_KEY'` with your actual API key, and this example will check the status of the API.
